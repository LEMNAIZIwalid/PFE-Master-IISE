# Software App (React Frontend + Express Backend)

## Structure
- `frontend/`: React + Vite
- `backend/`: Node.js + Express

## Full Setup
```bash
# Terminal 1 - Frontend
cd software-app/frontend
npm install
npm run dev
# http://localhost:5173

# Terminal 2 - Backend
cd software-app/backend
npm install
npm run dev
# http://localhost:5000
```
